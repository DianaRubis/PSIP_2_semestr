Демонстрация обработки событий в JavaScript

Пример 1: Атрибут HTML

Обработчик назначен прямо в разметке через атрибут onclick:

<input value="Нажми меня" onclick="alert('Клик!')" type="button">

Пример 2: Использование функции

Отдельная функция вызывается при клике:

function countRabbits() {
  for(let i=1; i<=3; i++) {
    alert("Кролик номер " + i);
  }
}

Пример 3: Свойство DOM-объекта

Обработчик назначается через свойство DOM-элемента:

elem.onclick = function() {
  alert('Спасибо');
};

Пример 4: Доступ через this

Внутри обработчика this ссылается на текущий элемент:

<button onclick="alert(this.innerHTML)">Нажми меня</button>

Пример 5: Метод addEventListener

Позволяет добавлять несколько обработчиков на одно событие:

elem.addEventListener("click", handler1);
elem.addEventListener("click", handler2);
elem.onclick = () => alert("Привет");
