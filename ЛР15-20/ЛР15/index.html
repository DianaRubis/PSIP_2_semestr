Объекты в JavaScript

1. Что понимается под объектом в JavaScript?

В JavaScript объект — это составной тип данных, который позволяет хранить коллекции свойств (пар ключ-значение) и методы (функции). Объекты используются для представления сложных структур данных.

// Примеры встроенных объектов JavaScript: // 1. Объект Math для математических операций // 2. Объект Date для работы с датами и временем // 3. Объект String для работы со строками // 4. Объект Array для работы с массивами // 5. Объект Console для вывода информации в консоль // 6. Объект Document для работы с DOM

2. Пример описания пользовательского объекта в JS

Пользовательские объекты можно создавать с помощью литерала объекта, конструктора или класса.

// Создание объекта с помощью литерала объекта let car = { brand: "Toyota", model: "Camry", year: 2020, color: "синий", startEngine: function() { return "Двигатель запущен!"; }, getInfo: function() { return `${this.brand} ${this.model}, ${this.year} года, цвет: ${this.color}`; } };

3. Пример доступа к свойствам объекта в JS

Доступ к свойствам объекта можно получить двумя способами: через точку и через квадратные скобки.

let person = { name: "Иван", age: 30, "favorite color": "синий" // свойство с пробелом в названии }; // Доступ через точку console.log(person.name); // "Иван" // Доступ через квадратные скобки console.log(person["age"]); // 30 // Для свойств с пробелами или специальными символами // можно использовать только квадратные скобки console.log(person["favorite color"]); // "синий"

4. Команда для удаления свойства

Для удаления свойства из объекта используется оператор delete.

let book = { title: "JavaScript для начинающих", author: "Иванов И.И.", pages: 350, price: 1200 }; // Удаляем свойство price delete book.price; // Проверяем результат console.log("price" in book); // false console.log(book.price); // undefined

5. Команда для проверки существования свойства

Для проверки существования свойства в объекте используется оператор in или метод hasOwnProperty().

let student = { name: "Мария", age: 20, course: 2 }; // Проверка с помощью оператора in console.log("name" in student); // true console.log("grade" in student); // false // Проверка с помощью метода hasOwnProperty() console.log(student.hasOwnProperty("age")); // true console.log(student.hasOwnProperty("subject")); // false

6. Команда для перебора свойств объекта

Для перебора свойств объекта можно использовать цикл for...in или методы Object.keys(), Object.values() и Object.entries().

let phone = { brand: "Samsung", model: "Galaxy S21", storage: 128, color: "черный" }; // Перебор с помощью for...in for (let key in phone) { console.log(`${key}: ${phone[key]}`); } // Получение массива ключей с помощью Object.keys() let keys = Object.keys(phone); console.log(keys); // ["brand", "model", "storage", "color"] // Получение массива значений с помощью Object.values() let values = Object.values(phone); console.log(values); // ["Samsung", "Galaxy S21", 128, "черный"] // Получение массива пар [ключ, значение] с помощью Object.entries() let entries = Object.entries(phone); console.log(entries); // [["brand", "Samsung"], ["model", "Galaxy S21"], ...]
